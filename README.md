# Школа юного иподьякона

Асинхронный сайт на FastAPI с минималистичным интерфейсом, анимациями и кастомной серверной разметкой для учебных глав.

## Возможности

- список глав на главной странице;
- страница главы с якорным содержанием;
- серверный парсер разметки текста;
- всплывающие материалы для комментариев, изображений и YouTube;
- структура проекта, готовая к деплою на Railway.

## Локальный запуск

```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
uvicorn app.main:app --reload
```

Сайт будет доступен по адресу `http://127.0.0.1:8000`.

## Разметка текста

- `#жирный текст#` — акцентный жирный текст
- `~похилий тонкий текст~` — тонкий курсив
- `!!важный текст!!` — красный жирный важный текст
- `[[текст_ссылки|comment:Комментарий]]` — всплывающий комментарий
- `[[текст_ссылки|https://site/image.jpg]]` — всплывающее изображение
- `[[текст_ссылки|https://www.youtube.com/watch?v=...&t=43s]]` — всплывающее видео

## Деплой на Railway

1. Создай новый GitHub-репозиторий и загрузи туда проект.
2. В Railway выбери `New Project` -> `Deploy from GitHub repo`.
3. Подключи репозиторий с этим проектом.
4. Railway сам найдёт `requirements.txt`, `Procfile` и `railway.json`.
5. После сборки сервис запустится командой:

```bash
uvicorn app.main:app --host 0.0.0.0 --port $PORT
```
